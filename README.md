# MyPorto - Website Portofolio Rizki Firdaus Purnama

Ini adalah website portofolio pribadi yang dibuat untuk memamerkan proyek, keahlian, dan latar belakang Rizki Firdaus Purnama sebagai seorang mahasiswa Computer Science di Binus University dengan fokus pada Front-End Development.

**[https://rizkifirdauspurnama.github.io/MyPorto_Web/]** 


## ‚ú® Fitur Utama

* **Desain Responsif:** Dibangun menggunakan Bootstrap 5, website ini beradaptasi dengan baik di perangkat desktop, tablet, dan mobile.
* **Navbar Tetap (Sticky):** Menu navigasi utama tetap terlihat di bagian atas layar saat pengguna melakukan *scroll*.
* **Efek Ketik (Typing Effect):** Judul nama ("Rizki Firdaus Purnama") muncul dengan animasi mengetik saat halaman pertama kali dimuat.
* **Animasi Scroll (Reveal):** Elemen-elemen di berbagai bagian (seperti *Latest Project*, *Why Hire Me*, dan *Skills*) muncul dengan efek *fade-in* saat di-scroll.
* **Interaksi Hover:**
    * **Tombol "See Project":** Berubah warna (background transparan, teks berwarna) saat kursor mendekat.
    * **Ikon *Skills*:** Latar belakang ikon berubah menjadi putih saat kursor mendekat.
* **Detail Proyek:** Setiap kartu proyek dilengkapi dengan *tags* teknologi yang digunakan dan tombol "See Project" yang mengarah ke tautan eksternal.
* **Tautan Sosial Media:** Ikon yang terhubung langsung ke profil GitHub, Instagram, dan LinkedIn.
* **Download CV:** Tombol fungsional untuk mengunduh file CV.
